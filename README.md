# Create-a-Sales-Analytics-Dashboard

## Power BI Sales Analytics Dashboard

This project presents an **interactive Sales Analytics Dashboard** built using **Power BI** to analyze sales performance, shipment trends, and key business metrics.
The dashboard helps businesses track revenue trends, product performance, and regional sales insights to support **data-driven decision making**.

## Key Features

* Sales performance tracking across different regions
* Product category analysis and revenue contribution
* Shipment and order trend monitoring
* KPI tracking including total sales and order volume
* Interactive filters for deeper business insights

---

## Business Insights

The dashboard enables stakeholders to:

* Identify top-performing products and regions
* Monitor sales growth trends
* Analyze shipment performance
* Track key business KPIs in real time
